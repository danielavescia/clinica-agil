# Projeto: SISTEMA DE MARCAÇÃO DE CONSULTAS

# Descrição: O projeto consiste no desenvolvimento de um sistema de marcação de consultas. 
O objetivo é permitir que um usuário possa criar perfis de pacientes, marcar e cancelar  consultas médicas.
O sistema será iniciado com dados de paciente.

# Requisitos
versão: .net 7.0


# Uso
 
3 - Execute o ClinicaConsultas.exe (bin\Release\net7.0\ClinicaConsultas.exe)

OU 

Compile o programa na IDE e clique em RUN;

# Sistema de Gerenciamento de Despesas - Menu

O programa fornece um menu interativo para o sistema de gerenciamento de marcação de consultas. Ela permite ao usuário escolher diferentes opções, como cadastrar paciente, marcar consulta e desmarcar consultas.


## Utilização

1. Execute o programa.
2. Será exibido um menu inicial com as opções disponíveis.
3. Digite o número correspondente à opção desejada e pressione Enter.
4. Siga as instruções exibidas na tela para interagir com o sistema.
5. Ao concluir uma operação, o programa retornará ao menu inicial para que você possa escolher outras opções ou sair do sistema.
